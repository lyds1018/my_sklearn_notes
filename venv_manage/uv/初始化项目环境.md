## 🚀 uv 初始化项目环境

### 1. 初始化

```
 uv init my_project
```

生成一个名为 `my_project` 的目录，并生成 `pyproject.toml`

### 2. 进入目录

```
 cd my_project
```

### 3. 创建虚拟环境

```
 uv venv --python 3.10
```

### 4. 激活虚拟环境

- **Linux / macOS**

  ```
   source .venv/bin/activate
  ```

- **Windows PowerShell**

```
 .venv\Scripts\activate
```

### 5. 安装依赖

- 手动添加依赖（会写入 `pyproject.toml` 和 `uv.lock`）：

```
 uv add numpy pandas
```

- 根据项目要求一次性添加依赖：

```
 uv sync
 uv add -r requirements.txt
```
