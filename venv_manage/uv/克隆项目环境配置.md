# 🛠 uv 配置克隆项目环境

## 一、进入项目目录

```shell
  cd my_project
```

---

## 二、项目含 **pyproject.toml**：

```Shell
  uv sync
```
- 自动解析 `pyproject.toml` 和 `uv.lock`，并安装依赖

---
## 三、项目无**pyproject.toml**：
### 1. 创建虚拟环境

```python
  uv venv --python 3.10
```
- 根据项目要求选择 python 版本

### 2. 激活虚拟环境

- **Linux / macOS**:

```shell
  source .venv/bin/activate
```

- **Windows (PowerShell)**:

```shell
  .venv\Scripts\activate
```

### 3. 安装依赖

```shell
  uv add pandas==2.2.0
  uv add -r requirements.txt
```
